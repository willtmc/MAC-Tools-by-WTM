Tutorial: How to Use the QR Code Generator

  • Enter Auction Details: Input your auction code, starting lot number, and ending lot number.
  • Generate QR Codes: Click the "Generate PDF" button. The tool will create a PDF with QR code stickers for each lot in the specified range.
  • Print and Use: Download and print the PDF using Avery 5960 labels. Each sticker includes a QR code linking to the lot on the McLemore Auction website, the lot number, and the auction website URL.
