Supported CSV Formats

1. CRS Tax Assessor Format (Recommended)

You can directly upload the CSV export from CRS without any modifications. The system will automatically process the following columns:

  • Owner 1 - Will be used as the recipient name
  • Owner Address - Mailing address
  • Owner City - City
  • Owner State - State
  • Owner Zip - ZIP code

2. Simple Format

For manually prepared data, ensure your CSV has these exact column names:

  • Name - Full name of recipient
  • Address - Street address
  • City - City name
  • State - Two-letter state code
  • Zip - 5 or 9 digit ZIP code

Processing Notes:

  • Names will be truncated to 40 characters
  • Addresses will be standardized using USPS format
  • Duplicate addresses will be removed
